Odoo Sales Target for Salesman Apps

Total Sales Target for Sales Person Odoo apps helps to manage sales target for salesperson , user can create and track sale target for each salesperson and also send email to a particular sale person with targeted details. You can make sales target for all sales person based on specific date internal which helps to generate sales target for monthly, quarterly or for any specific date interval. We have provided option to achieve target based on sales order confirmation, delivery order, invoice creation or while invoice is paid. When any sales order is created it also shows achieved target, total target and difference for target for selected salesman on sales order.
